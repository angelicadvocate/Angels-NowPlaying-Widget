Angels NowPlaying Widget

Instructions (Work in Progress)

Welcome! This page will guide you through setting up and using the NowPlaying Widget with OBS and Tuna.

Setup Steps

  1. Download the entire repository from the GitHub page.
  2. Unzip if needed and place the Angels-NowPlaying-Widget folder in the location where you want to keep the widget files on your computer.
  3. Install and configure Tuna inside OBS.
  4. Configure Tuna to output the files Artist.txt, Track.txt, and Artwork.png into the root directory of the widget folder. Some template files have already been added in the root directory so that you will know where to place them.
  5. Open the built-in overlay editor by clicking the Edit buttons for each template in the editor dashboard. The color provided in the frame editor will match the color to use for the progress bar in obs.
  6. To add a widget to your OBS scene, create a new Browser Source, select the Local File option, and point it to the corresponding HTML file for your chosen template.
  7. Set the resolution in OBS to match the dimensions shown on the editor page for the selected template.
  8. Adjust the Browser Source settings as needed, such as enabling Shutdown source when not visible to save resources.
  9. Ensure the widget is positioned correctly in your OBS scene, and test it by playing audio through Tuna.
  10. You can color match the progress bar by adding the tuna progress bar source in OBS and setting the color to match what the template editor listed on the screen. Layer the progress bar behind the transparent portion of the widget to complete the effect.

Important Notes

Local File Playback: This widget has been tested primarily with local audio files played via VLC integration inside OBS with Tuna v1.9.9. To use local music, add your music directory as a VLC Video Source in OBS (not a Media Source). Only VLC Video Sources currently work with Tuna for local playback.
Current Status: Only Template 1 is fully functional at the moment. Other templates are under development and may not render correctly.

Future Features
